# MCP

## What This Category Means

MCP covers Model Context Protocol servers and related integration patterns that let AI tools connect to external systems, data, and workflows.

## Why It Matters

MCP servers can turn an AI assistant from a text-only helper into a practical engineering operator with controlled access to files, repositories, browsers, databases, and product tools.

## What Will Be Collected Here

- Official and community MCP server candidates.
- Evaluation notes.
- Installation checklists.
- Permission and security guidance.
- Tested use cases and recommended configurations.

## How To Evaluate Entries

Evaluate each MCP server by source, maintainer quality, permissions, token requirements, setup complexity, security risks, local testability, and rollback path.
