# Official MCP Servers

## Purpose

Track official or high-confidence MCP server candidates for future research, evaluation, and installation planning.

This file is not a full recommendation list yet.

## Selection Criteria

- Official source or clearly maintained repository.
- Active maintenance and clear documentation.
- Practical value for AI engineering workflows.
- Understandable permission model.
- Clear installation, configuration, test, and removal process.

## Priority Levels

- P0: install first
- P1: high value
- P2: useful later
- P3: optional
