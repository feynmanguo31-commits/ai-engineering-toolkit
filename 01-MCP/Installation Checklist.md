# MCP Installation Checklist

Use this checklist before installing or configuring any MCP server.

- [ ] Confirm source.
- [ ] Confirm maintainer.
- [ ] Confirm required permissions.
- [ ] Confirm tokens needed.
- [ ] Confirm least-privilege setup.
- [ ] Confirm local test command.
- [ ] Confirm rollback/removal method.
- [ ] Confirm security notes.
