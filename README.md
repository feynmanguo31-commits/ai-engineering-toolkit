# AI Engineering Toolkit

AI Engineering Toolkit is a curated, practice-oriented toolkit for building AI products with modern agent workflows.

This repository collects, tests, documents, and organizes practical AI engineering tools and workflows so they can be reused with confidence.

## What This Repo Is For

- Collecting practical tools, patterns, and workflows for AI engineering.
- Testing candidates before treating them as recommended.
- Documenting setup, tradeoffs, permissions, risks, and use cases.
- Reusing proven MCP servers, Agents, Subagents, Skills, Hooks, Goal / Loop patterns, prompts, templates, and best practices.

## What This Repo Is Not

- Not a random bookmark folder.
- Not a list of trendy tools without testing.
- Not a place to store secrets, tokens, credentials, or private keys.

## Usage Principle

collect -> test -> document -> reuse

Every entry should move through this path before it becomes part of the recommended toolkit.

## Safety Principle

- Use least privilege for tools, agents, and integrations.
- Do not store secrets in this repository.
- Prefer official or actively maintained tools.
- Document risks before installation or configuration.
