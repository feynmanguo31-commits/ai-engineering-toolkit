# Agents

## What This Category Means

Agents are AI systems or configurations that can plan, act, use tools, and pursue engineering tasks across multiple steps.

## Why It Matters

Well-designed agents can reduce repeated manual work, improve consistency, and make complex AI product development workflows easier to execute.

## What Will Be Collected Here

- Agent design patterns.
- Role definitions.
- Tool-use boundaries.
- Evaluation notes.
- Tested agent workflows.

## How To Evaluate Entries

Evaluate agents by task fit, reliability, tool safety, observability, failure modes, handoff behavior, and measurable usefulness.
