# Skills

## What This Category Means

Skills are reusable instructions, workflows, or capability packages that help an AI assistant perform a specific type of task consistently.

## Why It Matters

Skills preserve useful process knowledge and make repeated AI engineering work easier to execute without rediscovering the same steps each time.

## What Will Be Collected Here

- Skill ideas.
- Skill specifications.
- Usage notes.
- Evaluation examples.
- Patterns for creating and maintaining skills.

## How To Evaluate Entries

Evaluate skills by clarity, repeatability, scope, triggering conditions, required tools, maintenance burden, and quality of outputs.
